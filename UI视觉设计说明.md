- **参考**：商城分类/商品 → 朴朴；个人中心/订单 → 淘宝
- **布局**：移动端 375×812（iPhone 标准）
- **圆角**：卡片 12px，按钮 22px（胶囊）
- **底部导航**：5 Tab 固定

## 各页面要点

### 1. 首页

- 顶部定位：「师大自提站」
- 搜索栏 + 3 展位轮播
- 积分快捷入口：签到 / 做任务 / 邀请好友
- 分类宫格 + 热门推荐商品双列流
- 底部 5 Tab 导航

### 2. 我的

- 用户信息区：头像、昵称、等级、积分、提现、开通会员（剩余天数）
- 我的订单：待收货 / 已收货，展示自提点与到站时间
- 9 项功能菜单列表

### 3. 商品详情

- 会员价 + 积分兑换双价格展示
- 自提点与预计到站时间
- 三种支付方式选择：纯积分 / 现金 / 积分+现金
- 底部「加入购物车」「立即购买」

### 4. 赚积分

- 当前积分余额
- 7 日签到日历
- 任务列表（广告、分享、完善资料等）
- 邀请好友 Banner

### 5. 确认订单

- 自提点选择卡片
- 商品清单 + 配送费明细（≤100 元收 3 元）
- 支付方式切换
- 底部合计与提交订单

### 6. 订单详情（核销码）

- 订单状态时间轴
- 居中核销二维码（参考抖音核销模式）
- 自提点地址与预计到站时间
