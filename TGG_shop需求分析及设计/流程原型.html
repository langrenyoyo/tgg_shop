TGG Shop 流程原型

版本 v1.2 · 2026-06-22 · 对齐《需求文档 v1.2》

开屏广告 页面导航 购物下单 配送时段 自提淘宝模式 做任务API 签到 月度排行榜 提现 代理核销 代理申请

0. 开屏广告

打开 APP 展示，后台可关闭

flowchart LR A[打开APP] --> B{开屏广告开关} B -->|开启| C[展示广告3-5秒] C --> D{可跳过?} D -->|是| E[倒计时后跳过] D -->|否| F[等待结束] B -->|关闭| G[直接进入首页] E --> G F --> G

1. 页面导航架构

flowchart TB Splash[开屏广告] --> Home[首页] Home --> Points[赚积分] Home --> Checkout[确认订单] Points --> AdSign[签到看广告页] Points --> Tasks[任务列表] Checkout --> Cutoff[配送日提示弹窗] Mine[我的] --> Withdraw[提现] Mine --> CS[客服] Mine --> AgentApply[申请代理→客服]

2. 购物下单与履约

自提/送货上门双开关 · 截单配送日 · 自提仅核销

flowchart TD A[进入结算] --> B{自提开关} B --> C{送货上门开关} C --> D[选择配送方式] D --> E[站点自提] D --> F[送货上门] E --> G[选自提点+联系地址] F --> H[选收货地址] G --> I[计算配送费] H --> I I --> J[展示预计配送日] J --> K[提交订单] K --> L{配送方式} L -->|自提| M[生成核销码] M --> N[代理扫码核销] L -->|送货上门| O[待发货→配送中→已送达]

4. 站点自提（淘宝模式）

用户可填联系地址 · 提货到自提点 · 地址后台同步

flowchart TD A[用户管理联系地址] --> B[下单选自提点] B --> C[后台配置自提点详细地址] C --> D[用户端实时同步展示] D --> E[到站出示核销码] E --> F[代理扫码核销提货]

5. 做任务（悬赏 API 对接）

task_type → task_list → task_info → task_register → get_examine_list

flowchart TD A[做任务 Tab] --> B[task_type 分类] B --> C[task_list 列表] C --> D[task_info 详情] D --> E[按 option 填表+upload] E --> F[task_register 提交] F --> G[get_examine_list] G --> H[回调发积分]

6. 签到（独立板块）

flowchart TD A[签到 Tab] --> B[签满30天送XXX] B --> C[看广告签到] C --> D[连续天数+1]

7. 积分排行榜月度结算

flowchart TD A[积分入账] --> B[累计当月积分] B --> C[实时排名展示] C --> D{每月1日00:00} D -->|是| E[定榜发奖→清零] D -->|否| B

7. 提现流程

1元起 · 手续费1% · 提到微信 · 后台可配置

flowchart LR A[我的-提现] --> B[输入金额≥1元] B --> C[扣除1%手续费] C --> D[提现至微信] D --> E[写入提现记录]

8. 代理核销仅自提

自提点仅核销，无其他功能

sequenceDiagram participant U as 用户 participant A as 代理 U->>A: 出示核销码 A->>A: 扫一扫核销 A->>U: 核销成功提货

9. 申请代理

跳转客服，非表单自助提交

flowchart LR A[申请点位代理] --> B[跳转客服] B --> C[客服沟通审核] C --> D[后台标记代理] D --> E[开通扫一扫]
